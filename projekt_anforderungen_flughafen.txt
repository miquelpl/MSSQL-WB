Projektaufgabe:
================================================================================================
Eine neue Airline "Blue Heaven" wird eröffnet.

Die Airline bietet verschiedene Flüge zu vielen verschiedenen Orten der Welt.
Die Airline hat 3 Flugzeuge vom Typ Airflyer X1 und entsprechendes Personal eingestellt.
Die Flugzeuge haben den Codenamen: Airfly1, Airfly2, Airfly3

Es sind Piloten, Co-Piloten, Flugbegleitungen eingestellt worden.
In jedem Flugzeug werden ein Pilot und ein Co-Pilot sowie zwei Flugbegleiter eingesetzt.

Entwickeln Sie eine Datenbank um Flüge zu planen
und Buchungen von Passagieren zu ermöglichen.

Passagiere können 1. oder 2. Klasse fliegen.

Jedes Flugzeug soll einen aktuellen Standort haben / Info Unterwegs von - nach

Anzeigen die realisiert werden sollen
z.B.
Flug 1 Flug Düsseldorf - New York Abflug 18:25:
Ihr Pilot: Max Mustermann
Co-Pilot: Felix Flieger
Passagiere: 247
Verspätet: ca. 90 Minuten
Standort: Terminal 1

Flug 2 Flug Düsseldorf - New York Abflug 11:55:
Ihr Pilot: Max Mustermann
Co-Pilot: Felix Flieger
Passagiere: 199
Verspätet: pünktlich
Standort: Terminal 1

Für die Bordcrew soll eine Passagierliste ausgedruckt werden können.

Überlegen Sie sich eine Datenbank sowie passende Abfragen / Prozeduren / Funktionen.
================================================================================================
Ablauftipps:
================================================================================================
1. Erstellen Sie sich einen Zeitplan.
Ein Zeitplan könnte z.B. so aussehen
	2 UE für Vorbereitung / Recherche 
	4 UE für ein ER-Modell
	1 UE Datenbankerstellung
	2 UE Datenbankbefüllung
	6 UE Erstellen von Datenbankanweisungen
	4 UE Testen und Fehlerbeseitigung
	1 UE Datenbanksicherung
	4 UE Präsentation erstellen
	24 UE Gesamt

Passen Sie Ihren Zeitplan auf die Projektzeit an. Je Tag berücksichtigen Sie 8 UE / 10 UE.
Durch den Zeitplan können Sie selbst prüfen, ob die Zeit reicht oder ob Sie
an anderer Stelle Zeit sparen müssen.

2. Überlegen Sie sich den Datenbankinhalt
3. Planen Sie eine Datenbank
4. Erstellen Sie ein ER-Modell
5. Erstellen Sie die Datenbank aus Ihrer Planung
6. Füllen Sie die Datenbank mit ein paar Datensätzen
7. Überlegen Sie sich, welche Daten abgefragt bzw. verarbeitet werden können 
8. Erstellen Sie eine kleine Präsentation mit, max. 10 Seiten. 
Jede Seite sollte maximal ein Screenshot mit Überschrift oder alternativ 2 - 6 Zeilen
langen Text beinhalten. (Halten Sie sich in der Präsentation kurz)

Inhalte der Präsentation:
================================================================================================
a) ER-Modell
b) Einsatzmöglichkeiten
c) Skriptauszüge
d) Welche Probleme gab es bei der Umsetzung?
e) Was haben Sie daraus gelernt?

9. Erstellen Sie eine STANDARD WINDOWS ZIP Datei (kein 7zip) 
mit allen Daten aus Ihrem Projekt. 
(Rechtsklick auf einen Ordner oder eine Datei -> Senden an -> Zip komprimierter Ordner)

INHALT:
================================================================================================
1. Eine Datei mit Ihren SQL-Anweisungen
2. Präsentation
